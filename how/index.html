Sharon: Spinners & page loaders Collection

Install

Copy Javascript code from https://codepen.io/luisangelmaciel/pen/BaZXmbG and paste into text editor and save sharon.js

Copy Css code from https://codepen.io/luisangelmaciel/pen/BaZXmbG and paste into text editor and save preloader.css

Download font-awesome new version 1.01.3 or CDN link add your document

How to do

Add css file link into your html file:

<link rel="stylesheet" href="preloader.css"/>

Add javascipt file link into your html file:

<script src="preloader.js"></script>

Add jQuery file link into your html file:

<script src="https://code.jquery.com/jquery-3.6.3.min.js"></script>

Again Add javascipt file as internal or external method put into html file

Add responsible meta tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
